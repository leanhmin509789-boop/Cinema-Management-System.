# 🎭 Hệ Thống Quản Lý Rạp Chiếu Phim (Teatro)

Dự án C++ cấu trúc dữ liệu và giải thuật mức độ 3 sao ⭐⭐⭐.

## 🛠 Cấu Trúc Dữ Liệu Sử Dụng
- **BST (Binary Search Tree)**: Quản lý phim, tìm kiếm phim nhanh chóng với độ phức tạp $O(\log n)$.
- **2D Array**: Mô phỏng sơ đồ ghế ngồi trực quan, xử lý đặt vé, hủy vé và đổi vị trí ghế.
- **Queue**: Quản lý hàng chờ hoàn vé theo cơ chế FIFO (Vào trước ra trước).
- **Stack**: Lưu lịch sử đặt vé gần nhất, hỗ trợ tính năng Hoàn tác (Undo tối đa 5 lần).

## 🚀 Chức Năng Chính
1. Quản lý danh sách phim (Thêm/Xóa/Tìm kiếm).
2. Đặt vé, hủy vé và đổi vị trí ghế trên ma trận 2D.
3. Hàng chờ duyệt hoàn tiền vé tự động.
4. Thống kê doanh thu chi tiết theo từng bộ phim.
5. Xuất báo cáo dữ liệu định dạng file `.txt`.
